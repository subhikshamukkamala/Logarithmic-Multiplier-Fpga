# Logarithmic Multiplier using Radix-4 Booth Encoding on FPGA

## Overview
This project presents the design and implementation of a **logarithmic multiplier** using **Radix-4 Booth Encoding** on an FPGA. The goal is to improve multiplication speed and reduce hardware complexity compared to traditional methods.

## Features
- Efficient multiplication using Radix-4 Booth Encoding  
- Reduced number of partial products  
- Logarithmic approximation for faster computation  
- Implemented in **Verilog HDL**  
- Tested on **Xilinx Artix-7 FPGA**

## Tools and Technologies
- **Verilog HDL**  
- **Xilinx Vivado**  
- **FPGA:** Artix-7  


## How It Works
The design uses **Booth encoding** to reduce partial products during multiplication. A **logarithmic approach** is applied to improve speed and efficiency. The system is synthesized and tested on FPGA for validation.

## Results
The design shows improved performance in terms of **speed** and **resource utilization** compared to conventional multipliers.

## Applications
- Digital Signal Processing (DSP)  
- Image Processing  
- Machine Learning Accelerators  

## Conclusion
This project demonstrates an **efficient FPGA-based multiplier design** suitable for **high-speed digital systems**.
